# TENeT Frontend

React + TypeScript frontend with Leaflet map visualization for Alaska's telecommunications network.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The app will be available at http://localhost:5173

## Tech Stack

- React 18
- TypeScript
- Vite
- Leaflet
- React-Leaflet

## Features

- Interactive Alaska map
- City markers with demographic information
- Bounded map view (restricted to Alaska region)
- Responsive design with header/footer
